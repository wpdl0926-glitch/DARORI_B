## 📌 Overview
돌봄 인프라가 갖춰진 마을은 살고 싶은 마을이 됩니다. 마을돌봄공동체를 운영하고 확산하며, 지역의 지속가능성을 실험합니다.

## 🤝 Team Rules
우리 팀의 원활한 협업을 위한 규칙입니다.
- 1. 소통 잘하기 (방향성, 성과 현황 등의 공유)
- 2. 처음 열정이 끝까지 가도록.
- 3. 부담 갖지 말고 의견 표현 열심히 하기.
- 4. 맡은 업무에 책임을 가지고 임하기. (기한 준수, 현황 업데이트)
- 5. 아이디어에 집중하기 전에 어떤 문제를 해결하고 싶은지 문제 정의에 노력을 기울이기.
- 6. 효율적인 시간 분배 (다양한 주제로 환기하며 프로젝트 완성도를 높이기)

- **Commit Message:** `[타입] 내용` 형식을 유지합니다. (예: `feat: 로그인 기능 추가`, `docs: 규칙 수정`)
